About

It was necessary to create a site with the ability to manage all pages, components and data through the administrative panel. Implement good SEO by optimizing images and code. Integrate third-party services to receive applications from the site. Create a blog for company articles.

Release

What has been done in this project.

Landings

I created about 10 landings from scratch: Launchpad, Tools, SafeMoon, GameF, DEX, Metaverse, Security Audit, Consulting, No Code Solutions, Rust, Marketing, Property Tokenization.

Blog plugin

Implementation of the blog in the project. Create different categories and specify articles for them. Conclusion of related articles. Convenient text editor. Indication of data for SEO, including links to twitter and original articles.

Portfolio plugin

Plugin for adding developed projects of the company in two sections: web development and mobile.

Feedback plugin

Plugin for working with applications from the pages of the site. In order to have access and not lose important applications - they are all stored in the CMS itself.

Template

A convenient site template was developed for editing and adding new components. All components are private, so they can be used anywhere on the site.

React -> CMS

The site was originally built with React. According to the task, it was necessary to make a website with an administrative panel. The project was completely transferred from the framework to Winter CMS. All styles, markup and functionality have been preserved.

Optimisation

Redesigned site styles and removed unnecessary classes. Updated page code. Tags and their classes were removed and optimized by name. All images have been optimized and minified.

CRM integration

Integration of a third-party CRM system to receive applications from the site. Each page has a name, so a registration system was made for applications indicating pages and certain data, without violating the general structure of the form on the pages.

SEO

Optimization of all project pages for SEO. Headlines, text, image captions and links have been redesigned. Search robots read all the data correctly due to additional information in the HEAD section and microdata.

Server

The server has been optimized for the project. Pages have caching and fast loading by increasing performance and disabling unnecessary server modules.

Images

Images of the project at the time of development. All pages are presented in original and finished form.
